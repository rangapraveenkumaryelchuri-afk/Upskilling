
# Local Community Event Portal

## Features

- HTML5 Semantic Structure
- CSS3 Responsive Design
- JavaScript Dynamic Events
- Bootstrap 5 Components
- jQuery Effects
- Geolocation
- Local Storage
- SQL Database Setup
- C# Console Setup

## Run Instructions

1. Open index.html using Live Server
2. Import SQL files into MySQL
3. Run C# project using:
   dotnet run
